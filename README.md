# Command Line Calculator using Node.js, TypeScript, and Inquirer

# SetUp :

# Clone repo
$ git clone https://github.com/Kanwal243/Typescript-Calculator-Task-Inquirer.git

# Navigate to project
$ cd typescript

# Install dependencies
$ npm i

# Run file
$ npm start
